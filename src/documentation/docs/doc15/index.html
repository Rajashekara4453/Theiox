    Web Scada

    Supervisory control and data acquisition (SCADA) is a control system architecture comprising computers, networked data communications and graphical user interfaces

    IoT Software comes with inbuilt web-based SCADA feature ,thats gives you centralized system to monitors and controls the entire area in your industry locally and remotely.

    Accessing Web-Scada

    To access Scada , click on Web SCADA on the component console

    • Click + Create

    • Type the name for your Scada board and click Save

    • Once click Save, it will connect you to the Scada Designing Console were all the necessary tools are available to design the layout as needed.

    Working with Scada

    Inside Scada Designing Board you have been provided with numbers of tools to help make your layout. in this section we will try to explore the features.

    Tools

    Top left tool bar , you have options for

    • Format Panel Helps to view the panels **seprately
    • Set the design board view Helps to set the view of the board in percentage, one page, two page etc..
    • Zoom helps in Zoom in and Zoom out the board
    • Undo and Redo Helps to Undo and Redo the edits
    • Delete - Delete Bin help to delete the seclected object
    • Arrange Helps to set the object postion to Front or Back
    • Fill Colour Helps to add the colours to the objects
    • Line Colour Helps to set the colours for the lines connection two objects
    • Shadow Helps to se the Shadow to the objects
    • Arrow Selection Helps to choose the different types of arrow as needed
    • Way points Helps to choose the different types of connections between the objects on the board.

    Left tool Bar

    • Search Search box help you to quickly search for the shapes, images you want use.
    • General In general section you have all Isometric shapes which you can select to create your diagram.
    • Flowchart helps you to select shapes that represnts the flow/ directions of the diagram.
    • Clipart Are pre-made images for you to use to illustrate the diagram
    • Images - Set of vector image to use in the diagram

    Top Right tool bar

    • Save Click on save will save the work
    • File Gives you option to Import, Export and to print
    • Edit helps you to Edit your design, Duplicate and also has the feature to Lock/Unlock the object
    • View Help ou to hide and show the tool bars, grids and guides ect..
    • Arrange - Helps to set the object postion to Front or Back, Rotate ,Flip the objects and Group the objects.

    Right tool

    Right tool bar is activated when the particular object is selected and shows the corresponding options for your slections

    • When you click on the empty space anywhere on the design board
      • Diagram selction are activated

    Under Diagram you have following feature View,Options and Paper Size

    • View Gives you option to see or hide the Grid, set the grid lines thickness and the grid colours.

    • Option
      Here you can enable and disable the connection arrows , connection points and Gudies for the objects for example if the Connection Arrow box is unchecked you will not be able to simply drag the arrow and connect it to other object. as shown in the image below the blue transprant arrow connectors will not be visible for the object

    • Paper Size

    helps you to set the papersize and orientation for your designs, Edit Dataand Clear Default Style

    • When you slect the objects on the design board , Right tool bar shows the corresponding options for your slections

    • Style Under style you set the backgound colour for the objects , add Border and set the border widt, you can also set the Opacity of the objects , edit image and its style.

    • Text Under Text you set all the related parameters for the text.

    • Arrange Under Text you set all the related parameters for the objects,like , The size, Position, Angle, Flip etc..

    Creating Diagram

    Now that you have understood the different tools and its working , let us now create our first diagram.

    • When scada is open you can see the design board , simply drage and drop the objects on the board and use connectors and flow chart to create your diagram.

    In the Image below we see a simple diagram created , it shows the enery geneartion , transmission and distribution .

    We can now configure the light bulb to see how much Power it is consuming .

    • It allows you to show the realtime value for the object and you can also set the benchmark/ threshold.

    The vaules can be shown in multiple colours, values are dynamic, when value crosses the threshold the colour changes as per set colours by you, eg. Red will display when the Bechmark vaule is crossed, it will show Orange when the vaule is getting close to Benchmark and shows Green when values are in control.

    Configuring the Diagram

    Here we will see how to configure an object in the diagram and set the vaules.

    • Right click on the object for which you would like to set the Value, and the click on Configure

    • Once you click on Configure It will take few seconds to load the console

    • Image below shows the configuration console, which has following elements to configure

    • Devices: In devices you can select the type of devices that images represents in the diagram eg Transformer Incomer. and it will display the vaules for it .

    • Tags : Tags are the parameter associated with a devices eg W Total

    • Decimals : You can use it to set the decimal, which show the vaules in decimals as per your selections.

    • Units : Units are Standard measure of a Parameter/Tag eg Watt etc..

    • Conditions : This is used to set the conditions for value represented , in condition you have the operands like >, <, = and !=*, with this operands you can apply the conditions to the vaules .

    • In condition you can also set the style for the vaule like Rotation, Fill colour etc.. In Creating Diagram section we talked about setting the Benchmark and the colours for the conditions , that is what you can achive with this settings.

    • you can click on + to add multiple conditions for the objects

    • Click Save to apply the conditions and then click Save again on the main console to update it on Scada.

    • Once your drawing and configuration is completed you can close the Web-Scada console by click on x on the brower tab.

    Viewing the Diagram

    • You can now see the Scada diagram in the IoT software. click on Web SCADA then click on the name of the diagram from the list

    • Now along with diagram you can see the values on the objects with you configure.you can perform following action on the diagram.
      • You can Zoom in and Out
      • you can Refresh
      • you can edit the diagram

    Editing the Diagram

    • Click on the Edit icon and it will open the diagram in the Scada design editor, make the necessary changes and click save your diagram will be updated.

    Deleting the Diagram

    • you can Delete, Share and Rename the diagram -Click on the Arrow beside the Diagram name in the title bar and click Delete to remove the diagram from the list.
