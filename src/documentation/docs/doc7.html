    Configuring Dashboard

    Setting-up New Dashboards

    To create new dashboards follow the steps given below,

    1. Click on Dashboard component

    Click (+)create to create a new Dashboard

    1. Enter the name for the dashboard in the Name textbox and click Save

    Associating Widgets to Dashboards

    Any number of widgets can be associated to a dashboard. The following steps must be performed to associate a widget to an existing Dashboard.

    1. Select the newly created dashboard or any one from the existing list for which you need to add the widgets.
    • Click on the +create button on the top right corner of the page and then click on Widget

    2.Select the type of Chart by which you need to represent the widget.

    After selecting the chart type , a new screen will apprear to configure the widget for your dashboard.

    • Widget Info : Here you will give the Title for the widget. eg "15mins Energy" depending what you want show on the dashboard.

    • Time Period : Here you need to select the time range to show the data . Note : There is an Overwrite Checkbox on the right corner. If the box is Checked then the Filters which we apply in the Dashboard overwrites. Else whatever we are applying filters while creating remains the same.. _ Select compare if you would like to compare the data with previous period.

    • Filter : In Filter you select the Type from where the Data should come from

    You have following slection under Filter Organization Hierarchy, Asset Group, Assets, and, Shift. ***

    After Making selection the items pertaining to the selection will appear, for example If you select Filter Type as Organization Hierarchy Items of Organization Hierarchy will be displayed. Based on the Hierarchy selected widgets will be displayed.

    • Tags : Under Tags you will be able to set the parameters to diaplay on your wigdets on dashboard. Based on Tag selected Unit will be displayed. Tag Section also give options to set Color, Y-axis Position, Frequency, Function, Benchmark and Threshold.
    • Note: You can add mulitple tags by click on Add Tag button.

    • Data Type : It is used for data representation.

    • Under Select Data Type, you have two options Category and Time series.

    • If you select Time Series then you have to select Time Stamp and Aggregation like Sum, Min, Max.

    • If you select Category then you have options to select tags as Group by liks Assets and the Aggregation like Sum, Min, Max.

    • Visualisation: which can be used for represenation in X-Axis like hide and show the Title by checking the box , placement of Date, select the Format of the date , you can set the widget Auo Refresh and also the Refresh time Interval , It also gives an option to set or restric the Decimal Value for the data, Show Legends and Grid lines forX and Y axis .

    Types of Widgets

    we have widgets for Historical Data and for Live Data.

    Under Historical we can use widgets like

    KPI

    KPI Is used to highlight any key metric in a dashboard for comprehension. The key metric can be accompanied with associated comparison indicator a target to show progression.

    Custom KPI

    It helps to make comparison with additional options like *Custom Multiplication Factor , Custom Unit Lable and Custom Value

    PF Chart

    It help you to measure of how effectively you are using electricity, with this widget you can display

    Alarm

    Help to view the real time Alerts on dashbaord.

    Tile

    A tile is a chart type that lets you select specific values from the cell to display.you can directly enter a cell intersection that provides the value for a tile.

    Table

    A table chart shows data arranged in rows and columns.

    Pie

    A pie chart is a statistical graphic, which is divided into slices to demonstrate numerical proportion.

    Stacked

    In a stacked column chart, data series are stacked one on top of the other in vertical columns, charts can show change over time and it's easy to compare total column lengths.

    Combined

    This widget help to combines two or more chart types in a single chart. To create a combination chart

    Line

    Line displays information as a series of data points called 'markers' connected by straight line segments.

    Bar

    It presents categorical data with rectangular bars, it is used to display and compare the number, frequency or other measure for different discrete categories of data

    MarkLine

    MarkLine is more informative. We can use MarkLines series by series, timeline by timeline, there is minimum and Maximum vaule filed in this widget where usercan set the mark. Minimum value should be "0" and Maximum vaule should be "75". beyond of the max value lines will be invisible.

    MarkPoint

    Markpoints makes a chart more informative. We can use it to add markPoints series by series, timeline by timeline. In this widge user can set the Min, Max and average points under the tags.

    Area

    It helps to displays graphically quantitative data.

    Scatter

    Helps to plot data points on a horizontal and a vertical axis to show how much one variable is affected by another.

    ON/OFF

    It give you the ability to remotely ON and OFF the physical Sensor device.

    Live Data Widget

    KPI

    This KPI can be used to represent live date

    Harmonics

    Helps to display non-linear loads and can be used to set thresholds and benchmarks

    Gauge

    It use needles to show information as a reading on a dial. On a gauge chart, the value for each needle is read against the colored data range and can be used to used in dashboard reports to show key indicators

    Table

    Horizontal Progress Bar

    ProgressBar is a graphical view indicator that shows selected parameters' progress.

    Vertical Progress Bar

    Same like Horizonatl ProgressBar is a graphical view indicator that shows selected parameters' progress in vertical form.
