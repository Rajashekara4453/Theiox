    Asset Control

    • Asset Control helps you with the asset usability within your organization; you can now schedule the ON/OFF function for assets like AC, Lights etc. Thus helping you to save energy during the non-working hrs of the Shift.

    Asset Control

    You can access the Asset Control by clicking Asset Control on the component console

    • Click on +Create to open Asset Control Configuration.

    • Note: Asset Groups and Tags used in this Asset Control configuration are created in their respective Module. For Example Tags can be created in Master Settings and Asset Group are created in Asset Management.
    • Under General Info Enter following information
      • Name
      • Groups
      • Description

    • Under Action Enter following information
      • Assets
      • Tag
      • Value : The Value is the Time in Minutes. (Time start is 00:00:00 hrs. If you would like to schedule Control time to 15hrs then you would need to Multiply 15x60 which is 900, so you will enter the Value as 900 in the Value filed)
      • Lable

    • Under Schedule Enter following information
      • Minutes
      • Hours
      • Type : Under Type you have option to schedule by Date, Day, and, Month (Depending upon your selctions you would need to select the range)
      • Month
      • Choose From and End Date

    In the Image below we have enter the schedule for 15mins and Hours is 3PM which is entered in 24hrs format.

    • Disable-Enable can be used to disable /enable the scheduling.
