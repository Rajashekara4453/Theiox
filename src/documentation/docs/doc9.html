    Setting up users account

    Adding New User

    To access User Accounts settings

    • Click Organisation Settings -> User Management

    To add new accounts follow the steps given below,

    • Click User Management -> User List

    • Inside User List click +create icon

    • Enter the Full Name, E-mail and, Phone Number in User Basic Info.
    • Note: Checking the Superuser box will grant user all the Administrator privileges.

    Creating User Access Level

    It is a set of permisison that restricts access to the modules in the software , These permission also govern user's right to View, Create, Edit and, Delete .

    Click User Management -> User Access Level

    To create predefined user access level follow this steps

    • Click + on create New, a new form will apprear on the right.

    • Enter the Name and Description

    • Under Permission you can set the permisison levels for all the available modules in software.

    Note : Clicking the Set Permission will Allow and Deny permisison at once for all the Module

    Creating User Work Group

    Work Groups are created to assign permissions at group level. User group is just a lable to identify to which group the user belongs to.

    • To create predefined work group under User Management click User Work Group then click on +Create Icon.

    • Enter the name and description and click save.

    • All groups created will appear on left side , which can be used when adding new users.

    Note: User work group , it just a lable to identify to which group that user belong and based on that we give him/her the access levels that governs the permissions

    User List

    User List shows the list of user created in the software. you can access it under User Management -> User List

    • User List is also used to edit the existing users details. Click on EDIT Button to edit.
    • To delete user record click Delete Button.

    My Profile

    • My Profile under User Management is basically for changing User Profile Picture, set new Theme colour and Change the password.

    • It also gives the information to the user like the Work Group he/she belongs and the Access Level of their account.
