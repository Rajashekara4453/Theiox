    Master Settings

    In Master Settings you would be able to create Tags, Units, Unit Converter, Multiplication Factors, Gate Types, and Sensor Types.

    Tags

    A Tag in IoT Software is referred to as any Parameter associated with a device.

    Adding Tag

    • To access Tag Configuration click on Master Settings ->Tags

    • Click on the +icon beside Create New.

    • Provide the Name of the Tag you want to add in the Tag Name field.

    • Provide a brief Description of the tag.

    • Provide the Unit of the Tag from the list. If the unit sought is not available in the list, the user can create one in the Units tab under Master Configuration.

    Editing Tag

    • Type the name of the Tag you want to edit in the Search for Tag field or click on the name of the Tag you want to edit from among the list of Tag names.
    • You can edit the Name, Description and/or the Unit of the Tag and then click on Update.

    Deleting Tag

    • Type the name of the Tag you want to delete in the Search for Tag" field or click on the name of the Tag you want to delete from among the list of Tag names. Click on Delete.

    Units

    A Unit refers to a standard measure of a Parameter/Tag.

    Adding Unit

    • Click + icon beside Create New.
    • Provide the Name of the unit to be added.
    • Provide the Alphabetical Symbol/ Notation relevant to the Unit.
    • Click on Save.

    Editing Unit

    • Type the Name of the Unit you want to update in the Search Unit field or click on the Name of the Unit you want to update from among the list of Unit names available.
    • Edit the Name and/or Notation fields as required.
    • Click on Update.

    Deleting Unit

    • Type the name of the Unit you want to delete in the Search Unit field or click on the name of the Unit you want to delete from among the list of Unit names available.
    • Click on Delete

    Unit Converter

    Unit Converter is used to manupulate the particular unit in higher or lower scale factor.

    • To access Unit Converter click on Master Settings ->Unit Converter

    Converting A Unit

    • Click on the From field. Select the unit you want to convert from the list. If the unit you are looking for is not available in the list, you can add the new unit in Unitssection.
    • Next click on the To field. Select the unit you want to convert to from the list. If the unit you are looking for is not available in the list, you can add the new unit in Unitssection..
    • Under the Value tab, Provide the scaling factor required to achieve the scaled value.
    • If you do not want to keep a conversion, you can delete the row on clicking the Delete Row.
    • Click on Add Row to add many more unit conversions.
    • Press the Enter key and you will be shown a pop up that the item is created and saved successfully.

    Multiplication Factor

    It is a factor used to multiply the reading of the energy meter to get the actual value consumed.

    • To access Multiplication Factor click on Master Settings ->Multiplication Factor

    Adding Multiplication Factor

    1. Click on the + Create New to add the Multiplication Factor.
    2. Enter Multiplication Factor Name, Description and Multiplication Factor Value.
    3. Click on the Save button to create a new Multiplication Factor.

    Editing Multiplication Factor

    • Type the name of the Multiplication Factor you want to update/modify in the Search Multiplication Factor field or click on the name of the Multiplication Factor you want to update/modify from among the list of Multiplication Factor names available.
    • Click on the Update button.
    • Click on the Delete button, if you need to delete the Multiplication Factor that is not required.

    Gateway Type

    Gateway Type is used to create Gateway in Master settings which is used in the Data Acquisition later.

    • To access Gateway Type click on Master Settings ->Gateway Type

    Adding Gateway Types

    • Click on the + Create New to create new Gateway type.
    • Enter the Name for the Gateway type to be created.
    1. Enter the ** Description** for the new Gateway Type.
    2. Click Save
    3. If you need to exit from the current page click on the Cancel button.

    Editing Gateway Types

    • Type the name of the Gateway Type you want to update/modify in the Gateway Types field or click on the name of the Gateway Types you want to update/modify from among the list of Gateway Types names available.
    • Click on the Update button.
    • Click on the Delete button, if you need to delete the Gateway Types that is not required.

    Sensor Types

    Sensor Types is used to create Sensor in Master settings which is used in the Data Acquisition later.

    • To access Sensor Types click on Master Settings ->Sensor Types

    Adding Sensor Types

    • Click on the + Create New to create new Sensor Types.
    • Enter the Name for the Sensor Types to be created.
    1. Enter the ** Description** for the new Sensor Types.
    2. Click Save
    3. If you need to exit from the current page click on the Cancel button.

    Editing Sensor Types

    • Type the name of the Sensor Types you want to update/modify in the Sensor Types field or click on the name of the Sensor Types you want to update/modify from among the list of Sensor Types names available.
    • Click on the Update button.
    • Click on the Delete button, if you need to delete the Sensor Types that is not required.
