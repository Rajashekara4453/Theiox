    Data Processing

    Data Processing Allows you to set Manual Entries , Batch Entries and create Tag Group . We will take a look at each one of it and see its configuration.

    Manual Entry

    Manual Entry is entering the parameters manually that cannot be fetched from the meters directly.

    • Click on the Add New to add the new Manual Entry for the Devices.

    • Enter Name in Manual Entry Name text box and a brief description in the Manual Entry Description text box.

    • Select the Devices based on Organization Hierarchy, Asset Group, Organization Hierarchyand Asset Group, and Assets

      • If you Choose Organization Hierarchy, select the Site and Level from Hierarchy.
      • If you Choose Asset Group, select Asset Group from the list.
      • If you Choose Organization Hierarchyand Asset Group, select both the Site and the level of Hierarchy and Asset Group from the list.
      • If you Choose Assets , select the list of Assets from the drop-down list.

    • Select the Tags based on Tags or Tag Group

      • If you Choose the Tag, Select the Tags from the list of Tags from the drop-down list.
      • If you Choose the Tag Group, Select the Tag Group from the drop-down list.

    • Select the Time for the Manual Entry to occur either shift wise or specific Time.

      • If you select the Time as Shift, Select the list of shifts from the drop-down list, which is from shift Management.
      • If you select the Time as Time, Enter the Pickup Time and click on Set.

    • Click on Save Config.

    Editing Manual Entry

    1. Select anyone from the Existing Manual Entries and click on the Edit icon on the left corner.
    2. Click on any of the Sites where changes have to be made and Select the Batch Entry if the Tags Based on Tag Group.
    3. Click on Submit Manual Entry.
    4. Now whenever you want the Data, Select the Pickup Time and click on Fetch, the Data for the Tags will be displayed.

    Batch Entry

    Entering a Group of Tags to execute as Batches. Tag Groups will be created in the Tag Group section separately. The created Tag Groups will be used herein forming a Batch Entry.

    Adding Batch Entry

    • Click on the add new to add new Batch Entry with the Existing Tag Groups.
    • Enter the Batch Entry Name, a brief description in the Description and add the Tag Groups from the list. Enter the Values for each tag under the Tag Group.
    • Click on Save once all the fields are filled. The new Batch entry will get created once the save button is clicked.

    Demo Video Batch Entry

    Tag Group

    One or more device parameters can be collected and formed to make a group named Tag Group. To Access Tag group Click Data Processing -> Tag Group

    Adding Tag Group

    • Tag Groups can be freshly created by clicking on + icon beside Add New.
    • Now under Add Tag Group Enter the name of the group you want to add. Next, provide a brief description of the Tag Group which indicates the purpose of creating such a Tag Group. Next, select all the parameters you want to include in the Tag Group by clicking on the drop-down button.

    Editing Tag Group

    • Type the name of the Tag Group you want to edit in the Search Tag Group field or click on the name of the Tag Group you want to edit from among the list of Group Tag names available on the list.
    • You can edit the Name, Description and/or the selection of Tags and then click on Update Tag Group.
